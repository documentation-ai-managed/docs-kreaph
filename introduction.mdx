## Overview

Kreaph provides a centralized space to organize, manage, and publish your project documentation. You create structured docs, collaborate with your team, and customize the experience to match your brand. Benefit from intuitive navigation, version control, and seamless integration with your workflows, making documentation maintenance effortless.

## Quick Start

Get started with Kreaph in minutes using these steps.

<Steps>
  <Step title="Create a Project" icon="plus">
    Log in to your dashboard at `https://dashboard.example.com` and click **New Project**.

    Name it `my-docs` and select your preferred template.
  </Step>
  <Step title="Add Your First Doc" icon="file-text">
    Navigate to the project and create a new MDX file.

````mdx
---
title: My First Page
description: Welcome to my docs.
---

## Hello World

This is your first documentation page.
````

  </Step>
  <Step title="Publish and Share" icon="globe">
    Preview changes, then publish. Share the live URL with your team.
  </Step>
</Steps>

<Callout kind="tip">
  Explore the full quickstart guide for advanced setup options.
</Callout>

## Brand Guidelines

Maintain consistency with Kreaph's brand elements. Use the primary color `#3B82F6` for accents, buttons, and links.

<Tabs>
  <Tab title="CSS Usage" icon="code">
    Apply the brand color in your custom theme.

````css
:root {
  --kreaph-brand: #3B82F6;
}

.button {
  background-color: var(--kreaph-brand);
  color: white;
}
````

  </Tab>
  <Tab title="Component Styling" icon="palette">
    Override defaults in your project settings.

````json
{
  "theme": {
    "primary": "#3B82F6",
    "headings": {
      "color": "#1E293B"
    }
  }
}
````

  </Tab>
</Tabs>
