## Creating Your First Project

Start organizing documentation by creating a project in kreaph.

1. Log in at `https://app.kreaph.com`.
2. Click "New Project" in the sidebar.
3. Enter your project name and description.
4. Click "Create" to finish.

Use your project dashboard to add and manage all your documentation easily.
